Buen día, este es mi proyecto para la prueba tecnica de fusepong, espero que les guste!.

- Como utilizarlo

Crea una cuenta desde la página de registro (no es necesario verificar el correo electrónico

1. Únete a una de las compañías ya creadas para acceder a los proyectos disponibles
2. Una vez dentro de un proyecto, podrás:
   - Ver y crear historias de usuario.
   - Crear y editar tickets. (estado, nombre, descripción)
   - Agregar comentarios a cada ticket.
